# Lost Cities

A multi-page archive exploring the world's lost and undiscovered archaeological sites — from Petra and Angkor Wat to Dholavira and Cahokia — inspired by National Geographic explorer Albert Lin's work using LiDAR, satellite remote sensing, and non-invasive archaeological technology.

**Live site:** https://manunibhavsar.github.io/lost-cities/

## Pages

- **Explore** — an interactive grid of lost cities, each with history, art, technology used in their (re)discovery, and associated myths
- **Technologies** — an overview of the remote-sensing tools (LiDAR, satellite radar, AI/ML classifiers, ground-penetrating radar) used to find these sites
- **About** — a timeline of Albert Lin's archaeological discoveries that inspired this project
